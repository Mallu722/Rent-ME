# Rent Me - Web Client

React web application for the Rent Me companion booking platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will run on `http://localhost:3000`

## Features

- User authentication (Login/Signup)
- Browse and search companions
- Create and manage bookings
- Real-time chat
- Payment processing
- Reviews and ratings
- Admin dashboard
- Companion profile management

## Tech Stack

- React 18
- React Router
- Axios
- Socket.IO Client
- Vite
